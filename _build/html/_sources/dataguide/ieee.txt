IEEE full text
==============

IEEE full text data is downloaded periodically onto Klabswing on weekends and is pushed to S3 storage on a 6month basis.  
This dataset is approximately 5.5TB as of Oct 2016.

Storage bucket
--------------

The data is currently stored on S3 Storage : s3://klabieee

Permissions
-----------

This data is only accessible from within the network and by users with the requisite access privileges.

Processed intermediates
-----------------------

This data has seen very little use so far. No intermediate datasets so far despite the vast and rich nature of this dataset.
