UChicago AURA¶

The Knowledge Lab has access to the grants database within the University of Chicago. The database dump from Feb 2016 is hosted within the Kotta, and is a restricted access dataset.

Storage bucket¶

The data is currently stored on S3 Storage : s3://klab-aura

Permissions¶

This data is only accessible from within the network and by users with the requisite access privileges.

Processed intermediates¶

None that are public.
